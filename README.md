# 💊 Smart Medicine Box (IoT Based)

## 📌 Overview
This project is an IoT-based smart medicine box designed to remind users to take medicines on time and monitor usage.

## 🚀 Features
- Automated medicine reminders
- Real-time monitoring using IoT
- Alert system using buzzer/LED
- Easy to use interface

## 🛠️ Technologies Used
- ESP32
- Embedded C
- IoT

## 🔌 Components
- ESP32
- Buzzer
- LEDs
- RTC Module
